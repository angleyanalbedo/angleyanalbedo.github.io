# 认真读文档即使是你熟知的领域
> 当我们在这个领域有所了解时，我们看文档就会习惯性的跳过去观看
> 但是这就会导致我们安装出错，安装程序也是如同写代码一般
> 在安装之后，再去排查错误是找不到的

### 不要随意的更改配置除非你清楚你在干什么

> 比如我在配置butterfly是
>
> 我将`_config.yml`
>
> ```yml
> filename_case: 0
> render_drafts: false
> post_asset_folder: true
> relative_link: true
> future: true
> ```
>
> `relative_link` 设置成`true`我以为没什么影响实际上这对于导航栏造成问题浪费了巨大时间
>
> 我还以为是`pjax`的影响
>
